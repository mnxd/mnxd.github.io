一、
名称：Atom sphere Scattering
简介：使用大气散射光照模型，天空盒中的太阳作为光源，模拟光线在大气中传播的效果
实现：从地面到外层空间的任何角度实时渲染大气，同时考虑瑞利和米氏多次散射。基于对所有视点、视点方向和太阳方向的预先计算的光传输方程的公式。
参考：网页：https://ebruneton.github.io/precomputed_atmospheric_scattering/
​ 论文1：http://www-ljk.imag.fr/Publications/Basilic/com.lmc.publi.PUBLI_Article@11e7cdda2f7_f64b69/article.pdf
​ 论文2：https://media.contentapi.ea.com/content/dam/eacom/frostbite/files/s2016-pbs-frostbite-sky-clouds-new.pdf

二、
名称：Auto Exposure
简介：天空盒中的太阳作为光源，使用自动曝光（人眼适应）模拟眼睛对照明条件的时间变化的反应
实现：计算场景的亮度几何平均值（log2平均值）并将其映射到某个“键值”，过曝光乘以所有像素，添加色调映射，颜色分级和伽玛。
参考：网页1（详细算法）：https://knarkowicz.wordpress.com/2016/01/09/automatic-exposure/
​ 网页2：https://docs.unrealengine.com/en-us/Engine/Rendering/PostProcessEffects/AutomaticExposure
​ 网页3：https://docs.unity3d.com/Manual/PostProcessing-EyeAdaptation.html
​ 网页4：https://github.com/Unity-Technologies/PostProcessing/wiki/Auto-Exposure
​ 辅助参考（HDR照明）：https://frictionalgames.blogspot.com/2012/09/tech-feature-hdr-lightning.html

三、
名称：Bloom Effect
简介：天空盒中的太阳作为光源，使用泛光效果使所有明亮区域产生光晕效果
实现：渲染一个有光场景，提取出场景的HDR颜色缓冲以及只有这个场景明亮区域可见的图片。被提取的带有亮度的图片接着被模糊，将结果添加到HDR场景上面
参考：https://learnopengl-cn.github.io/05%20Advanced%20Lighting/07%20Bloom/

四、
名称：Cascaded Shadow Map
简介：解决阴影贴图的透视混叠问题
实现：CSM的基本思想是将视锥体分成多个层级。为每个subfrustum渲染阴影贴图; 然后，像素着色器从地图中采样最接近所需分辨率的样本，这样在渲染大场景的阴影就可以避免传统的SM的各种缺点。
参考：理论：https://docs.microsoft.com/en-us/windows/desktop/dxtecharts/cascaded-shadow-maps
​ 论文：https://developer.download.nvidia.com/SDK/10.5/opengl/src/cascaded_shadow_maps/doc/cascaded_shadow_maps.pdf

八、
名称：Normal Map +Parallax Map
简介：NM:用向地面上附加纹理（法线贴图）的方式来模拟光照在凹凸不平表面的效果，增加额外细节，提升真实感

九、
名称：OIT
简介：场景中有两个不同的不透明物体（兔子），以及两个颜色不相同的半透明窗户，它们的渲染无关顺序
实现：使用排序后的片元链表来重新排序光栅化之后的片元，然后在片元着色器中完成融混操作。链表的每个元素都记录一个对最终像素颜色有贡献的片元值。屏幕的每个像素都会产生一个独立的链表，其中包含所有对这个像素有贡献的透明片元项。
参考：https://developer.download.nvidia.com/SDK/10/opengl/src/dual_depth_peeling/doc/DualDepthPeeling.pdf

十、
名称：PBR Effect
简介：使用基于物理的渲染模拟光线，场景中的兔子可以改变颜色、金属度和粗糙程度
实现：基于物理模拟光照，对渲染方程求解
参考：理论：https://learnopengl.com/PBR/Theory
​ 论文： http://blog.selfshadow.com/publications/s2013-shading-course/karis/s2013_pbs_epic_notes_v2.pdf
​ 资料合集：https://interplayoflight.wordpress.com/2013/12/30/readings-on-physically-based-rendering/

十一、
名称：Percentage Closer Soft Shadow
简介：实现物体不同光照角度/距离条件下的不同阴影表现
实现：生成精度自适应的软阴影。PCSS方法建立在标准shadow mapping的基础上：当在视线视角对每个像素着色时，PCSS返回一个浮点值表明这个像素的受阴影度（替换了普通shadow mapping的深度比较方法）。
参考：源码: http://developer.download.nvidia.com/whitepapers/2008/PCSS_Integration.pdf
​ 论文: http://developer.download.nvidia.com/shaderlibrary/docs/shadow_PCSS.pdf

十二、
名称：Post Process Volumetric Light
简介：一种考虑日光散射模型，体积遮挡方法的后处理方法，它能产生由大气阴影引起的体积光散射效应
参考：理论：https://developer.nvidia.com/gpugems/GPUGems3/gpugems3_ch13.html

十三、
名称：Reflection Cube Map
简介：镜面反射立方贴图
参考：论文：https://seblagarde.files.wordpress.com/2012/08/parallax_corrected_cubemap-siggraph2012.pdf

十四、
名称：Rendering_LLL
简介：实现透明几何光照和粒子效果的光照
实现：将动态光照存储在游戏场景中所有元素的每个像素可访问的一个链表中。屏幕上每个像素仅存储重叠的灯光，在相机空间中存储最小和最大深度。
参考：理论：GPU Pro 6 Chapter: Real-Time Lighting via Light Linked List
​ SIGGRAPH 2014 Course http://advances.realtimerendering.com/s2014/index.html
​ 辅助资料：红宝书OIT顺序无关的透明渲染 数据结构图例可当作重要参考
​ Github DX实现 https://github.com/wolfgangfengel/GPU-Pro-6/tree/master/03_Lighting

十五、
名称：SSAO
简介：利用屏幕空间环境光遮蔽来模拟光的散射，使场景中的兔子在模型的褶皱与孔洞处变暗
实现：对于铺屏四边形上的每一个片段，根据周边深度值计算一个遮蔽因子，使用遮蔽因子减少或者抵消片段的环境光照分量。遮蔽因子是通过采集片段周围球型核心(Kernel)的多个深度样本，并和当前片段深度值对比而得到的。高于片段深度值样本的个数就是我们想要的遮蔽因子。
参考：网页：https://learnopengl-cn.github.io/05%20Advanced%20Lighting/09%20SSAO/

十六、
名称：Temporal Anti Aliasing
简介：通过混叠的手段抗锯齿
实现：把样本分布到过去的N帧中去，然后每一帧从过去的N帧中取得样本信息然后Filter，达到N倍Super Sampling的效果。
参考：理论：High-Quality Temporal Super Sampling[Siggraph 2014]
​ Temporal Super Sampling and Antialiasing[2014]
​ Temporal Reprojection Anti-Aliasing in INSIDE[GDC][2016]
​ An Excursion in Temporal Super Sampling[GDC][2016]
​ Amortized Super Sampling[Siggraph Asia 2009]
